Setup:
npm run dev

routes:
Rotas:

- `POST - /tasks`
    -> criação usando body
    -> criação via csv:
        - É preciso existar uma pasta csv com o arquivo tasks.csv é necessário
        - caso não quiser usar a o .csv, é só apagar o arquivo dentro da pasta.
    
- `GET - /tasks`
    
- `PUT - /tasks/:id`
    
- `DELETE - /tasks/:id`
    
- `PATCH - /tasks/:id/complete`
